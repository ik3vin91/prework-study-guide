Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

    To create a new branch:

    1. In the Command Line, navigate to the prework-study-guide (project name) directory
    2.ENTER "git status" to check which branch you're currently in. you should be in the main branch. if not, switch to main by ENTERING "git checkout main"
    3. In the command line, pull down the latest main branch by ENTERING "git pull origin main".
    4. from the main branch, use the following Git Command to create a new branch named "feature/add-html". ENTER "git checkout -b feature/add-html". You should see the message "Switched to a new branch 'feature/add-html'". Confirm you are in the new branch by ENTERING "git status". You should see the following. "On branch feature/add-html. nothing to commit, working tree clean.

    To commit changes and merge to main branch:

    1. Confirm you are in the feature branch by ENTERING "git status".
    2. Before pushing to repo, double check that the local branch is in sync with the base branch in GitHub. Pull down any changes by ENTERING "git pull origin main.
    3. Push the changes to the remote branch by ENTERING "git push origin feature/starter-code"
    4. Go to GitHub. You will notice a message your branch had recent pushes. "Hit the Compare & pull request button". Commit and Merge. Your branch is now safe to delete.
